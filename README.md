# Xenon static file microservice

## Build:
Clone repository:
``` bash
git clone https://github.com/SeniorShadifer/xenon-static-file-microservice
```

Go to repository directory:
``` bash
cd xenon-static-file-microservice
```

Build microservice:
``` bash
make build
```

Run microservice:
``` bash
./static_file_microservice
```
